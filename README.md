# 🏥 AIA Insurance Tracker Update Management System

A desktop-based insurance management system developed using **JavaFX** to manage insurance policies, claims, and customer records efficiently.

This system was designed to help insurance companies streamline their operations by providing an organized platform for tracking policies, claims, and customer information.

---

## 🚀 Features

✔ Manage insurance policies
✔ Manage customer details
✔ Claims management system
✔ Feedback management
✔ Generate unique claim and feedback IDs
✔ User-friendly JavaFX interface
✔ Efficient data management with database integration

---

## 🛠 Technologies Used

**Frontend**

* JavaFX

**Backend**

* Java
* JDBC

* MySQL

* IntelliJ IDEA / Eclipse
* Git & GitHub

## 📂 Project Modules

### 👤 Customer Management

* Add new customers

### 📑 Policy Management

* Add insurance policies

### 💰 Claims Management
